# 📖 유저 가이드 — 헬스 기록 앱

처음 쓰는 분을 위한 화면별 사용법입니다. 전체 기능 목록은 [README](README.md)를 참고하세요.

---

## 1. 화면 구성

하단 탭 5개가 기본 화면입니다.

| 탭 | 하는 일 |
|---|---|
| 🏠 **홈** | 주간 목표 링 · 러닝 시작 · 오늘의 계획 · 진행 중 운동 복구 · 빠른 시작 |
| 🏋️ **루틴** | 내 루틴 목록 · 운동 프로그램(5×5 포함) 입구 |
| 📅 **달력** | 운동 기록/예정 일정 확인 · 운동 계획 만들기 |
| 📊 **통계** | 주간 빈도 · 부위별 세트 · 운동별 진행 그래프 · 유산소 요약 · 인바디 입구 |
| ⚙️ **설정** | 테마 · 단위 · 타이머 옵션 · 커스텀 운동 관리 |

모든 데이터는 **기기 안에만** 저장됩니다(서버·계정 없음).

---

## 2. 3분 만에 첫 운동

1. **루틴 탭 → + 새 루틴 만들기** → 이름 입력
2. **+ 운동 추가** → 부위/기구/브랜드로 필터하거나 검색해서 선택 → `N개 추가`
3. 운동마다 **세트 수 · 휴식 시간** 설정 (운동 시간은 스톱워치로 자동)
4. **루틴 저장** → 목록에서 **▶ 운동 시작**

> 더 빠르게: **루틴 탭 → 🔥 운동 프로그램 둘러보기**에서 검증된 루틴(맨몸 입문, 푸시/풀 데이 등)을 바로 가져올 수 있어요.

---

## 3. 운동 타이머 사용법

운동은 **준비 → 운동 → 휴식** 3단계로 진행됩니다.

- **준비**: `3·2·1·GO` 카운트다운(비프음) 후 자동으로 운동 시작
- **운동 중**: 스톱워치가 위로 올라갑니다. 다 했으면 **화면(타이머)을 탭** → 기록 모달
  - 모달에서 **몇 회 했는지** 입력 (근력) / **시간·거리** 확인 (유산소)
  - 무게는 **원판 개수 ±** 또는 **숫자 입력 → 원판 자동 배치** (바벨 그림 실시간 반영)
  - 목표 횟수 미달이면 **실패 원인**(무게/휴식/수면/통증/자세/집중)을 선택할 수 있어요
- **휴식**: 카운트다운 후 **자동으로 넘어가지 않습니다**. 준비되면 탭 → 다음 세트
  - 휴식 중 `휴식 시간 ±`로 이번 세트만 즉석 조절
  - 종료 3초 전 "곧 시작!" 경고(색+비프), 0초 후엔 초과 시간(+00:05) 표시

**세트표에서 할 수 있는 것**
- 세트 번호를 탭 → **W(워밍업) / D(드롭)** 지정 (워밍업은 볼륨·PR·증량 판단에서 제외)
- `+ 세트 추가 / − 마지막 세트`, 운동 메모
- **원판 계산기** 버튼: 목표 무게 → 한쪽에 끼울 원판 구성 그림
- **📈 무게 추이 그래프**: 지금 하는 운동의 세션별 **최고 무게(실선) + 추정 1RM(점선)**

**운동 완료 화면**
- 소요 시간·총 볼륨·완료 세트 요약, 🏆 **PR(신기록) 축하**
- **RPE(6~10)** 를 운동별로 기록 → 다음 자동 증량에 사용 (아래 참고)
- 앱을 도중에 꺼도 **자동 임시 저장** — 다시 열면 홈에서 이어하기/저장/삭제

## 4. 자동 증량 (RPE 기반)

같은 운동을 다시 시작하면 지난 기록을 보고 무게를 **자동으로 채워줍니다**.

| 지난 세션 결과 | 이번 무게 |
|---|---|
| 성공(모든 세트 목표 달성) + RPE 8 이하 | **+2.5kg** (lb 단위는 +5) |
| 성공 + RPE 9~10 | 같은 무게 유지 |
| 목표 미달/미완 | 같은 무게 재도전 |

RPE를 안 찍어도 성공이면 증량됩니다. 운동 화면 상단에 근거가 표시돼요(📈/😮‍💨/🔁).

## 5. 🏃 러닝 모드 (유산소)

**홈 → 러닝 시작** 카드로 진입합니다.

1. 종목 선택: 야외 러닝 · 걷기 · 등산(GPS 자동 거리) / 러닝머신 · 사이클 · 로잉머신(수동)
2. **시작** → 큰 거리(km) + 시간 · 평균 페이스 · 심박(워치 연결 시) 실시간 표시
   - 심박은 **존 색**으로 표시: Z1 회복 → Z2 지방연소 → Z3 유산소 → Z4 무산소 → Z5 최대
   - **1km마다 구간 스플릿** 자동 기록(구간별 심박 포함)
   - 실내 종목은 `±0.1km` 버튼으로 거리 입력
3. ⏸ 일시정지 → ▶ 재개 / ⏹ 종료
4. 요약(거리·시간·페이스·심박·칼로리·구간별 스플릿+존) 확인 → **기록 저장**

일반 루틴에 유산소 운동을 넣으면 세트표가 **시간(분)/거리(km)** 컬럼으로 바뀌고, 세트 완료 시 운동한 시간이 자동 기록됩니다.

## 6. 5×5 스트렝스 프로그램

**루틴 탭 → 운동 프로그램 → 스트롱리프트 5×5**

1. 최초 진입 시 **설정**: 운동 요일(기본 월/수/금) · 시작 루틴(A/B) · 운동별 시작 중량 · 휴식 시간
2. **대시보드**에서 오늘의 운동(A 또는 B)·일정·알림 확인 → **운동 시작** (권장 중량 자동 채움)
   - Workout A: 스쿼트/벤치/로우, Workout B: 스쿼트/OHP/데드리프트 — **완료 순서대로 A↔B 교차**
   - 운동을 놓쳐도 순서가 밀릴 뿐 꼬이지 않아요 (건너뜀 처리도 가능)
3. **중량은 자동 관리**: 5×5 성공 → +2.5(하체 일부 +5) / 실패 → 같은 무게 / **3연속 실패 → 10% 감량(deload) 제안**
4. **진행 통계 보기**: 리프트별 최고 중량·예상 1RM·성공률·성장 그래프·최근 실패 목록
5. 연속 이틀 운동 시 회복일 경고, 각 리프트의 **대체 운동 보기** 지원

---

## 7. 운동 계획 (플래너)

**달력 탭 → ➕ 운동 계획** (또는 홈의 오늘의 계획)

1. 루틴 선택 → 달력에서 **원하는 날짜들을 탭** (여러 날 선택 가능)
2. 저장하면 달력에 예정 표시: **예정 / 오늘 / 완료 / 놓침** 상태로 구분
3. 당일이 되면 홈의 **"오늘의 계획"** 카드에서 바로 시작

---

## 8. 기록 확인

- **달력**: 날짜별 세션(부위 색 마커). 세션 탭 → 상세에서 무게/횟수/시간/거리 수정, 유산소는 ♥심박·구간 스플릿도 표시
- **통계**: 주간/8주 빈도, 부위별 세트, **운동별 진행**(최고 무게/추정 1RM/볼륨 그래프), **유산소 요약**(횟수/시간/거리/평균 심박)
- **인바디 기록** (통계 → 인바디 카드): 키·체중·체지방률·골격근량 입력 → **BMI 자동 계산 + 분류**, 항목별 추이 그래프

---

## 9. 테마 & 설정

- **설정 → 테마**: 🖤 블랙(기본) · 🤍 화이트 · ⚙️ 메탈 · 💚 그린 · 🧈 버터
  - 웹은 즉시 적용, 앱(폰)은 완전히 종료 후 다시 열면 적용
- **설정**: kg/lb 단위 · 준비 카운트다운 · 진동 · 화면 켜짐 유지 · 세트별 RPE 입력칸 표시
- **커스텀 운동 관리**: 직접 만든 운동/기구 삭제
- **전체 초기화**: 모든 데이터 삭제 (되돌릴 수 없음!)

**나만의 기구 추가**: 운동 추가 화면 → **+ 직접 운동 만들기** → 이름/부위/기구 + **브랜드**(목록 선택 or 직접 입력). 새 브랜드는 필터에도 자동 추가됩니다.

## FAQ

**Q. 데이터를 다른 기기로 옮길 수 있나요?**
아직 백업/동기화 기능이 없습니다. 데이터는 기기(웹은 브라우저)별 로컬 저장입니다.

**Q. 운동 중 앱이 꺼졌어요.**
자동 저장됩니다. 앱을 다시 열면 홈 상단에 "진행 중이던 운동" 배너가 떠요 — 이어하기를 누르세요.

**Q. 소리가 안 나요.**
설정의 카운트다운 알림을 확인하세요. 웹은 브라우저 탭이 소리 허용 상태여야 하고, 폰에서는 무음 모드면 진동으로 대체됩니다.

**Q. 증량 추천이 이상해요.**
지난 세션에서 워밍업(W)으로 표시 안 한 세트가 미완이면 "재도전"으로 판단합니다. 세션 상세에서 기록을 수정하면 추천도 다시 계산돼요.
